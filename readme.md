# Мини приложение
Приложение с двумя страницами, где можно выбирать позиции меню.


# Проект приложение покупок
Приложение для просмотра сделок на основании мок данных, взятых через API.

## Описание проекта
Проект на TypeScript с использованием препроцессора SASS. Первая страница имеет приветствие и кнопку (а также ссылку) для перехода на вторую страницу, которая включает в себя разделы с различными меню, которые можно выделять, наблюдая суммарное количество и стоимость. <br>В рамках одной сессии выделенные данные во всех разделах сохраняются, если выйти на первую страницу (перезагрузить или нажать "вернуться на первую страницу"), данные будут сброшены.</br>

## Структура проекта
1. `/index.html` — разметка сайта, позиционирования элементов
2. `/src/main.ts` — основной файл рендера и отображения всех данных
3. `/src/menu-state.ts` — функции взаимодействия с данными (никакой связи с DOM)
4. `/src/data/menus.js` — датасет с данными для наполнения меню (сгенерированы ИИ)
3. `/fonts/*.css` — шрифт, используемый на сайте
5. `/images/logo.ico` — иконка сайта
6. `/styles/styles.*` — файлы стилей sass и скомпилированный из него css
7. `/dist/*.*` — скомпилированные JavaScript файлы
